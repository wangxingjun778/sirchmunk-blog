Sirchmunk's Web UI provides a unified interface for real-time chat with streaming search logs, source citations, knowledge cluster browsing, and system monitoring — all with dark/light theme support and bilingual localization.

**Features:**
- Streaming chat with inline search logs
- File-based RAG and web-augmented search modes
- Knowledge cluster analytics
- System health and LLM usage monitoring
- Dark/light theme with English and Chinese localization

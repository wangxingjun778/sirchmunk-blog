Sirchmunk 通过模型上下文协议（MCP）暴露其智能搜索能力，实现零摩擦的 AI 到 AI 通信。任何兼容 MCP 的 AI 助手都可以将 Sirchmunk 的搜索作为原生工具调用。

**功能特性：**
- stdio 和 HTTP 传输模式
- 兼容 Claude Desktop、Cursor IDE 等
- 搜索、目录扫描和知识管理作为 MCP 工具
- 通过 `mcp_config.json` 轻松配置
